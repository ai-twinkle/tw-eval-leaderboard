## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly. We appreciate your help in keeping our users safe.

### How to Report

- **Preferred Method**: Use GitHub's private vulnerability reporting feature if available (check the "Security" tab in the repository).
- **Alternative**: Open a private issue or contact the maintainers directly at [maintainers@ai-twinkle.com](mailto:maintainers@ai-twinkle.com) or via GitHub issues.

### What to Include

When reporting a vulnerability, please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact and severity
- Any suggested fixes or mitigations

### Our Response

- We will acknowledge receipt of your report within 48 hours
- We will provide a more detailed response within 7 days indicating our next steps
- We will keep you informed of our progress throughout the process
- Once the vulnerability is resolved, we will publicly disclose it after giving you credit (if desired)

## Security Considerations

This is a static website for visualizing AI benchmark data. While we take security seriously, please note:

- The application runs entirely client-side
- No user data is stored on our servers
- Uploaded files are processed locally in the browser
- We recommend using this tool with trusted data sources

Thank you for helping keep our project secure!
